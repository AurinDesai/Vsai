# CodeForge Studio - Super Quick Start

## Start in ONE Command

### Windows (PowerShell)
```powershell
python start_codeforge.py
```

### Mac/Linux (Terminal)
```bash
python3 start_codeforge.py
```

### Or Double-Click (Windows)
```
start_codeforge.bat
```

---

## What Happens

1. ✅ Server starts automatically
2. ✅ Browser opens automatically  
3. ✅ You're ready to code!

---

## First Time Use

1. **Set Auth Token**: Click button → Enter `localdev`
2. **Create File**: File → New File → Type code → Ctrl+S
3. **Search Files**: Press Ctrl+P
4. **Upload Folder**: File → Add Existing Folder (Upload)
5. **Split Editor**: Click "Split Right" button

---

## Key Shortcuts

- **Ctrl+P** - Find file
- **Ctrl+S** - Save file
- **Ctrl+/** - Comment line

---

You now have a VS Code-like editor with file management, git integration, and optional AI. No installation needed!
